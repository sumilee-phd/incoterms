# 인코텀즈 조건 추천 실무 앱

무역 실무 초급자를 위한 **Incoterms 2020 조건 추천·비교 교육용 웹앱**입니다. 수출입 거래 조건을 입력하면 적합한 인코텀즈 조건을 추천하고, `FOB`, `CFR`, `CIF` 등 주요 조건의 차이를 한 화면에서 비교할 수 있습니다.

## 바로가기

- 배포 앱: https://incoterms-smoky.vercel.app/
- GitHub Repository: https://github.com/sumilee-phd/incoterms

## 수업 활용 목적

이 앱은 무역 실무 입문자가 인코텀즈를 암기식으로 접근하기보다, 거래 조건을 바꾸면 추천 조건이 어떻게 달라지는지 직접 확인하도록 만든 교육용 실습 도구입니다.

주요 학습 포인트는 다음과 같습니다.

- 운송비 부담 주체에 따라 `FOB`, `CFR`, `CIF`가 어떻게 달라지는지 이해
- 보험 가입 여부가 `CFR`과 `CIF`를 구분하는 핵심임을 확인
- 해상운송 조건과 복합운송 조건의 차이 확인
- 11개 Incoterms 2020 조건을 E/F/C/D 그룹으로 비교
- 조건 대조와 퀴즈를 통해 수업 내용을 즉시 복습

## 5분 핵심 시연 순서

수업 시간이 짧을 때는 아래 순서만 진행하면 됩니다.

| 순서 | 화면 | 조작 | 설명 포인트 |
|---|---|---|---|
| 1 | 맞춤 추천 | 기본값 그대로 확인 | 해상 수출에서 판매자가 운송비와 보험을 부담하면 `CIF`가 추천됨 |
| 2 | 맞춤 추천 | 보험 가입을 `아니오`로 변경 | 보험이 빠지면 `CIF`가 아니라 `CFR`에 가까워짐 |
| 3 | 맞춤 추천 | 국제 운송비 부담을 `아니오`로 변경 | 구매자가 운송을 맡으면 `FOB` 또는 `FCA` 계열로 이동 |
| 4 | 조건 대조 | `FOB`와 `CIF` 비교 | 두 조건 모두 위험 이전은 선적 시점이지만, 운송비·보험 부담이 다름 |
| 5 | 전체 비교표 | E/F/C/D 그룹 확인 | 인코텀즈 11개 조건을 책임 범위별로 묶어 설명 |
| 6 | 기초 퀴즈 | 1문항 풀이 | 학습 내용을 바로 점검 |

## 10분 확장 시연 가이드

### 1. 앱 접속 및 화면 소개

배포 앱에 접속합니다.

https://incoterms-smoky.vercel.app/

강사 설명 예시:

> 이 앱은 인코텀즈 조건을 외우는 도구가 아니라, 거래 조건을 선택하면 어떤 조건이 적합한지 추천받고 비교해 보는 실습용 앱입니다.

확인할 화면 요소:

- 상단 탭: `맞춤 추천`, `조건 대조`, `전체 비교표`, `기초 퀴즈`
- 왼쪽: 거래 조건 입력
- 오른쪽: 추천 조건과 판매자·구매자 부담 항목

### 2. 기본값으로 CIF 추천 확인

`맞춤 추천` 탭에서 기본 선택값을 확인합니다.

기본 시연 조건:

- 거래 방향: 수출
- 운송 방식: 해상운송
- 판매자가 국제 운송비 부담: 예
- 판매자가 운송 보험 가입: 예
- 판매자가 수입통관까지 책임: 아니오

강사 설명 예시:

> 판매자가 목적항까지 가는 운송비와 보험을 부담하지만, 수입통관은 구매자가 처리하는 해상무역 조건이므로 CIF가 추천됩니다.

핵심 설명:

- `CIF = Cost, Insurance and Freight`
- 판매자: 수출통관, 목적항까지 운송비, 보험 부담
- 구매자: 수입통관, 목적항 이후 비용 부담
- 위험 이전: 선적항에서 본선 적재 시점

### 3. CIF와 CFR 차이 보여주기

`판매자가 운송 보험을 가입하는가?` 항목을 `아니오`로 변경합니다.

강사 설명 예시:

> 운송비는 판매자가 부담하지만 보험을 판매자가 가입하지 않으면 CIF가 아니라 CFR에 가까워집니다. CIF와 CFR의 가장 큰 차이는 보험입니다.

비교 포인트:

- `CFR`: 판매자가 운송비 부담, 보험은 구매자 부담
- `CIF`: 판매자가 운송비와 보험 모두 부담

### 4. FOB 조건으로 흐름 바꾸기

`판매자가 국제 운송비를 부담하는가?` 항목을 `아니오`로 변경합니다.

강사 설명 예시:

> 구매자가 국제 운송을 직접 준비한다면 판매자의 책임은 선적항에서 물품을 본선에 싣는 단계까지로 줄어듭니다. 이때 해상운송에서는 FOB 조건을 자주 비교합니다.

비교 포인트:

- `FOB`: 국제 운송비와 보험을 구매자가 준비
- `CFR`: 운송비는 판매자, 보험은 구매자
- `CIF`: 운송비와 보험 모두 판매자

### 5. 조건 대조 탭 시연

`조건 대조` 탭으로 이동합니다.

추천 비교:

- `FOB` vs `CIF`
- `CFR` vs `CIF`
- `EXW` vs `DDP`

강사 설명 예시:

> FOB와 CIF는 둘 다 해상운송에서 많이 보지만, 비용 부담 구조가 다릅니다. 앱의 조건 대조 화면을 보면 위험 이전, 운송비, 보험, 통관 책임을 한 번에 비교할 수 있습니다.

### 6. 전체 비교표 탭 시연

`전체 비교표` 탭으로 이동합니다.

설명 순서:

1. `E조건`: 판매자 책임 최소
2. `F조건`: 주운송비 구매자 부담
3. `C조건`: 주운송비 판매자 부담
4. `D조건`: 도착지 인도, 판매자 책임 확대

강사 설명 예시:

> 인코텀즈 11개를 처음부터 하나씩 외우기보다, E/F/C/D 그룹으로 나누면 판매자의 책임이 어떻게 커지는지 이해하기 쉽습니다.

### 7. 기초 퀴즈로 마무리

`기초 퀴즈` 탭으로 이동해 1~2문항만 풉니다.

추천 문항:

- CIF에서 보험을 준비하는 사람은?
- CFR과 CIF의 가장 큰 차이는?
- 판매자 책임이 가장 큰 조건은?

강사 설명 예시:

> 방금 본 추천 화면과 비교표를 기준으로 바로 퀴즈를 풀어보면, 조건 차이가 기억에 더 오래 남습니다.

## 수업 중 강조할 핵심 문장

- `CFR`과 `CIF`의 차이는 보험입니다.
- `FOB`, `CFR`, `CIF`는 해상운송 중심 조건입니다.
- `CPT`, `CIP`는 복합운송에서 함께 비교하면 좋습니다.
- 비용 부담 시점과 위험 이전 시점은 항상 같은 것이 아닙니다.
- 실제 계약 전에는 운송, 보험, 통관 책임을 계약서에서 반드시 확인해야 합니다.
